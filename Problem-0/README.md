# Find a pair with maximum product in array of Integers

Given an array with both +ive and -ive integers, return a pair with the highest product.

Examples :  

Input: arr = [1, 4, 3, 6, 7, 0]
Output: [6,7]
Output: 42  

Input: arr = [-1, -3, -4, 2, 0, -5]
Output: [-4,-5]
Output: 20

### Constrain
- arr.length >= 2
